Cincinnati Elastic User Group

The Cincinnati Elastic User Group is a group of developers, project managers, and data analysts who learn, experiment, and grow in a recruiter-free and sales-free zone. With a variety of speakers and workshops, new perspectives will be brought to your Elastic implementations and data analysis ideas. We focus on the free components of the Elastic stack, to ensure any attendee will walk away with a new skill that can be implemented in a project at no cost. (www.elastic.co/products)

Upcoming and Past Meetups

Take a look below at our past meetups. We're always looking for ideas for additional meetup topics and for additional speakers. If you have an idea for a topic or are interested in speaking at the meetup, send an email to info@sevenhillstechnology.com

Distributed Tracing with APM and Microservices

June 27th, 2019

For this meeting we're going to do a presentation and workshop on Distributed Tracing using APM. Bring your computers if you like and we'll have a sample github repo to clone that will be ready to set up with multiple microservices and we'll look at how distributed tracing is set up and can help you track performance metrics across a distributed architecture.

Special thanks to The Space Community Workplace for hosting (https://www.cincinnatispaces.com/)

Getting Data into The Elastic Stack.

February 28th, 2019

Given the numerous ways to ingest data into Elastic for reporting, monitoring, analytics, and full text search, it can be a challenging task to decide where to start.

This talk will review things to consider when importing data, and the various methods to get data into Elastic, and how to handle the data once it's there. Discussion topics will include programmer APIs/SDKs, Beats, Logstash, Ingest Pipelines, and third party options.

Intro to The Elastic Stack and APM

November 29th, 2018

Part 1: Give me all the search - An introduction to the Elastic stack and it's core functionality, use cases, and placement in software projects.

Part 2: Instrumenting a NodeJS application using Elastic APM. This talk will cover what APM is, how it's supported in Elastic, and how to get started tracking and monitoring a NodeJS application.

Enhancing Search

September 27th, 2018

Enhancing Search with Recommendations Search is a hard problem. Elastic search takes the power of the inverted index and enables developers to easily implement a basic search solution for their domain. However the demands and expectations on search will keep us on a quest for better results.

In this session we will learn about several approaches to enhancing search with personalized recommendations.
